Introduction

The palmetto data set contains growth and survival data for two dominant palmetto species, Serenoa repens and Sabal etonia, across multiple habitats, years, and experimental treatments. Data was recorded at the Archbold Biological Station located in Florida. This project highlights differences in height, canopy length, canopy width, and green leaves for the two species. A binary logistic regression model is used to see the odds of predicting correctly or incorrectly if a plant species is either the Sabal etonia or Serenoa repens based on plant height, canopy length, canopy width, and green leaves count.

Binary Logistic Regression

The next step in the analysis is to utilize a binary logistic regression model. The binary logistic regression model will be used to see the probability of predicting correctly if a plant species is either the Sabal etonia or Serenoa repens based on plant height, canopy length, canopy width, and green leaves count.

Show code 
palmetto_blr <- glm(species ~ length + 
                              width + green_lvs + height, 
                            data = palmetto,
                            family = "binomial")

#model fitting 

blr_fitted <- palmetto_blr %>% 
  broom::augment(type.predict = "response") %>% 
  mutate(predicted = ifelse(.fitted >= .5, "Serenoa repens", "Sabal etonia")) %>% 
  mutate(correct = ifelse(predicted == species, "predicted_correctly", "predicted_incorrectly"))
  
palmetto_table <- blr_fitted %>% 
  count(species, correct) %>% 
pivot_wider(names_from = correct, values_from = n) %>% 
mutate(total = rowSums(across(where(is.numeric)))) %>%
mutate(percent = (predicted_correctly / total)* 100) %>%
 mutate_if(is.numeric,funs(as.character(signif(.,3))))


kableExtra::kable(palmetto_table,
                  col.names= c("Species", "Correct Prediction",
                               "Incorrect Prediction", "Total", "% Correct")) %>% 
    kableExtra::kable_styling(bootstrap_options = "striped",
                            full_width = F,
                            position ="center") %>% 
  kableExtra::add_header_above(bold = TRUE,
                               line = TRUE,
                               c("Percent species predicted correctly by binary logistic regression model" = 5))

Summary

Based on the binary logistic regression model, the odds of predicting the Sabal etonia based on the plant characteristics of height, canopy length, canopy width, and green leaves count is 92.6%, and the odds of predicting the Serenoa repens is 90.8%.
